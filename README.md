# Быстрый тест Telegram-рейтинга прямо сейчас

Тестовый режим показывает текущее общее количество репостов и не выдаёт его за дневной прирост 10:00–22:00. Рабочий baseline при тестовом запуске не изменяется.

1. Создайте Telegram StringSession:

   ```bash
   python3 scripts/create_telegram_session.py
   ```

2. Добавьте в GitHub Secrets `TELEGRAM_API_ID`, `TELEGRAM_API_HASH` и `TELEGRAM_SESSION`.
3. Откройте **GitHub → Actions → Update Telegram Top 10 → Run workflow**.
4. Выберите `test_mode = true` и нажмите **Run workflow**.
5. После успешного запуска откройте сайт. Появится тестовый top-10 по текущему общему количеству репостов.
6. После теста снова запустите workflow с `test_mode = false` или дождитесь следующего автоматического запуска. Система вернётся к обычному режиму 10:00–22:00 по времени Тбилиси.
